Flag Defination:
isDownloaded - 1: data has been downloaded (converted Table)
isProcessed - 1: data has been settled

For beftn there are 5 flags:
isProcessedIncentive - when incentive has processed 
isProcessedMain - when main has processed 
isProcessed - when both main and incentive processed
isDownloaded - when both main and incentive processed

Type parameters definition in report table:
type 1 - Online
type 2 - Account Payee
type 3 - BEFTN
type 4 - COC
